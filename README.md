# Emby NFO Fixer

**Emby NFO Fixer** — Это утилита для проверки, исправления и автоматического дополнения метаданных в NFO-файлах Kodi для использования в медиасервере Emby.

## Возможности 

* **Автоматическое исправление тегов:** проверка структурированности и валидности метаданных в NFO.
* **Загрузка дат премьер (TMDB API):** если в NFO отсутствует тег `<premiered>`, утилита считывает `<tmdbid>` и автоматически запрашивает дату премьеры с The Movie Database (TMDB).
* **Безопасность данных:** создание резервной копии оригинальных файлов перед модификацией с сохранением структуры каталогов.
* **Детальное логирование:** раздельные журналы операций (`changed.log`, `skipped.log`, `error.log`) в папке резервных копий.

## Установка

### Готовые бинарные файлы (Рекомендуется)

В разделе **[Releases](https://github.com/alexls74/emby_nfo_fixer/releases)** скачайте архив для вашей операционной системы:
* **Windows:** `emby_nfo_fixer_vXX.XX.XX_windows_x64.zip` (x64)
* **macOS:** `emby_nfo_fixer_vXX.XX.XX_macOS_universal.zip` (Universal Binary)
* **Linux:** `emby_nfo_fixer_vXX.XX.XX_linux_amd64.tar.gz` (amd64)

Распакуйте архив в удобную папку.

### Сборка из исходного кода

* Скачайте и установите **[Go](https://go.dev/doc/install)**
* Скачайте исходный код Emby NFO Fixer
```bash
git clone https://github.com/alexls74/emby_nfo_fixer.git && cd emby_nfo_fixer.git
```
* Скомпилируйте приложение
```bash
make build
```
Готовые исполняемые файлы будут расположены в папке **_/build_**.

## Настройка TMDB API

Для автоматического добавления отсутствующей даты премьеры утилита использует данные **The Movie Database (TMDB)**.

1. При первом запуске утилита предложит ввести TMDB API токен.
2. Введённый токен будет проверен и сохранён в файл `emby_nfo_fixer.conf` рядом с исполняемым файлом.
3. Если вы пропустите ввод или укажете пустое значение, функция запросов к TMDB отключится. В дальнейшем можно внести вручную.

> **Как получить токен TMDB:**
> 1. Зарегистрируйтесь на сайте **[The Movie Database (TMDB)](https://www.themoviedb.org/)**.
> 2. Перейдите в **Profile and Settings → API Subscription**.
> 3. Создайте API ключ.
> 4. Скопируйте **API Read Access Token**.

## Использование

Запуск осуществляется из командной строки с обязательным указанием путей к папке с NFO-файлами и папке резервного копирования.

Пример:

```bash
emby_nfo_fixer /mnt/Movies /mnt/Backup
```
