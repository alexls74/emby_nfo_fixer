# Emby NFO Fixer

**Emby NFO Fixer** — Это утилита для проверки, исправления и автоматического дополнения метаданных в NFO-файлах Kodi для использования в медиасервере Emby.

## Возможности 

* **Автоматическое исправление тегов:** проверка структурированности и валидности метаданных в NFO-файлах.
* **Загрузка дат премьер (TMDB API):** если в NFO-файле отсутствует тег `<premiered>`, утилита считывает `<tmdbid>` и автоматически запрашивает дату премьеры с The Movie Database (TMDB).
* **Сканирования медиатеки:** если были внесены изменения в NFO-файлы, опционально запускается сканирование медиатеки Emby.
* **Безопасность данных:** создание резервной копии оригинальных файлов перед модификацией с сохранением структуры каталогов.
* **Детальное логирование:** раздельные журналы операций (`changed.log`, `skipped.log`, `error.log`) в папке резервных копий.
* **"Тихий" режим:** запуск без вывода информации в консоль.

## Установка

### Готовые бинарные файлы (Рекомендуется)

В разделе **[Releases](https://github.com/alexls74/emby_nfo_fixer/releases)** скачайте архив для вашей операционной системы:
* **Windows:** `emby_nfo_fixer_vXX.XX.XX_windows_x64.zip` (x64)
* **macOS:** `emby_nfo_fixer_vXX.XX.XX_macOS_universal.zip` (Universal Binary)
* **Linux:** `emby_nfo_fixer_vXX.XX.XX_linux_amd64.tar.gz` (amd64)

Распакуйте архив в удобную папку.

### Сборка из исходного кода

* Скачайте и установите **[Go](https://go.dev/doc/install)**
* Скачайте исходный код Emby NFO Fixer
```bash
git clone https://github.com/alexls74/emby_nfo_fixer.git && cd emby_nfo_fixer.git
```
* Скомпилируйте приложение
```bash
make build
```
Готовые исполняемые файлы будут расположены в папке **_/build_**.

## Настройка TMDB API

Для автоматического добавления отсутствующей даты премьеры утилита использует данные **The Movie Database (TMDB)**.

1. При первом запуске будет предложено ввести TMDB API токен.
2. Введённый токен будет проверен и сохранён в файл `emby_nfo_fixer.conf` рядом с исполняемым файлом.
3. Если пропустите ввод или укажете пустое значение, функция запросов к TMDB отключится. В дальнейшем можно внести вручную.

> **Как получить токен TMDB:**
> 1. Зарегистрируйтесь на сайте **[The Movie Database (TMDB)](https://www.themoviedb.org/)**.
> 2. Перейдите в **Profile and Settings → API Subscription**.
> 3. Создайте API ключ.
> 4. Скопируйте **API Read Access Token**.

## Настройка Emby API

Для автоматического запуска сканирования медиатеки Emby используется **[REST API](https://dev.emby.media/doc/restapi/index.html)**.

1. Запуск с ключом **[-e]** требует обязательного указания источника и папки резервного копирования.
2. При первом запуске будет предложено ввести данные сервера Emby.
3. URL сервера может быть как внутрисетевым (**`http://192.168.1.199:8096`**), так и внешним (**`https://domain.tld`**).
4. Введённые URL и API Key будут сохранёны в файл `emby_nfo_fixer.conf` рядом с исполняемым файлом.

> **Как получить токен Emby API Key:**
> 1. Зайдите в администрирование сервера Emby.
> 2. Перейдите в раздел **API-ключи**.
> 3. Создайте новый ключ.

## Использование

Запуск осуществляется из командной строки с обязательным указанием путей к папке с NFO-файлами и папке резервного копирования.

Пример без дополнительных опций:
```bash
emby_nfo_fixer /mnt/Movies /mnt/Backup
```
С запуском сканирования медиатеки Emby после внесения изменений в NFO-файлы:
```bash
emby_nfo_fixer -e /mnt/Movies /mnt/Backup
```
Режим без вывода информации в консоль:

```bash
emby_nfo_fixer -s /mnt/Movies /mnt/Backup
```
